### Como rodar?

```shell
git clone https://github.com/FusRoDah061/emaflix.git

npm i

# Subindo simultaneamente backend e frontend
npm run dev

# Acesse a aplicação em http://localhost:3000
# Acesse a API em http://localhost:3001
```
